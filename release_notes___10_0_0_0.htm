    Release Notes - 10.0.0

    v10.0.0 Release Notes

    AspDotNetStorefront v10.0.0 includes updates which have a very serious (and positive) impact on security. Please read the section in this Help console called 'Security and Tech' very carefully to make sure that you implement the application to take full effect of the changes.

     

    This is a major release, which justifies the move from AspDotNetStorefront 9.x to 10.x

    • STORE OWNERS: ... for when you use AspDotNetStorefront to sell successfully online, right out of the box
    • SYSTEM ADMINISTRATORS: ... for when you use AspDotNetStorefront as a store-builder, working on the presentation layer
    • DEVELOPERS: ... for when you use AspDotNetStorefront and source code to engineer your own functionality
